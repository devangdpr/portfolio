# 🚀 Personal Portfolio Website

A stunning, modern portfolio website with amazing graphics, gravity particle effects, and smooth animations.

## ✨ Features

- **Gravity Particle System**: Interactive particles with physics-based movement
- **3D Tilt Effects**: Cards that respond to mouse movement
- **Smooth Animations**: Fade-in, slide-up, and parallax effects
- **Glitch Text Effect**: Eye-catching hero section
- **Typewriter Animation**: Dynamic text display
- **Responsive Design**: Works perfectly on all devices
- **Interactive Navigation**: Smooth scrolling and mobile menu
- **Animated Skill Bars**: Progress bars that animate on scroll
- **Contact Form**: Functional contact section
- **Mouse Trail Effect**: Custom cursor with trailing dots

## 🌐 How to Deploy (Make it Live)

### Option 1: GitHub Pages (FREE & Easy)

1. **Create a GitHub account** at https://github.com
2. **Create a new repository** named `your-username.github.io`
3. **Upload your files**:
   - Click "uploading an existing file"
   - Drag and drop: `index.html`, `styles.css`, `script.js`, `README.md`
   - Click "Commit changes"
4. **Go live**: Visit `https://your-username.github.io`

### Option 2: Netlify (FREE & Fast)

1. Go to https://netlify.com and sign up
2. Drag and drop your project folder
3. Your site is live instantly!
4. Get a free URL like `your-site.netlify.app`

### Option 3: Vercel (FREE & Professional)

1. Go to https://vercel.com and sign up
2. Click "New Project"
3. Import your GitHub repository or upload files
4. Deploy with one click!

### Option 4: Traditional Web Hosting

1. Get hosting from providers like:
   - Hostinger
   - Bluehost
   - SiteGround
2. Upload files via FTP
3. Point your domain to the hosting
